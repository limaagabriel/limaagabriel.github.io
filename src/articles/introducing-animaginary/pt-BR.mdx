Quando você está construindo um site para uma empresa tão ambiciosa quanto a Planetaria, precisa causar impressão. Eu queria que as pessoas visitassem nosso site e vissem animações mais realistas do que a própria realidade.

Para tornar isso possível, precisávamos extrair cada gota de desempenho possível do navegador. E assim nasceu o Animaginary.

```js
import { animate } from '@planetaria/animaginary'

export function MyComponent({ open, children }) {
  return (
    <animate.div
      in={open}
      animateFrom="opacity-0 scale-95"
      animateTo="opacity-100 scale-100"
      duration={350}
    >
      {children}
    </animate.div>
  )
}
```

O Animaginary é nossa nova biblioteca de animação web que redefine o que você pensava ser possível na web. Escrita à mão em WASM otimizado, o Animaginary consegue até animar a propriedade `height` de um elemento a 60fps.
